# Sahir Vhora

Personal landing page at [sahirvhora.github.io](https://sahirvhora.github.io)

Open-source SAP SuccessFactors tools for consultants - config audit, position integrity, object sync, pay transparency, release tracking, and more.

## Tools

- **SF Compass** - 94 feasibility scenarios across 8 SF modules
- **Release Tracker** - Live tracker for 1H/2H SAP SF release changes
- **Position Integrity Checker** - Validate position data against foundation objects
- **ObjectSync** - Sync OM foundation objects PRD to Dev via OData v2
- **Config Compare** - Compare metadata and picklists across tenants
- **Impact Brief** - Personalised release impact briefs
- **Pay Transparency** - EU Pay Transparency Directive readiness
- **Value Navigator** - Consulting framework for value realisation
- **Agent Skills** - AI skills for SF consultants

Built with intent. No trackers. No sign-ups.
